CS 207 Lab Test v.3

Test Policies

  • This test is open book, and open web.
    • You are allowed to refer to your previous lab assignments, online forums, the Arduino web site, or the Sparkfun Danger Shield product page to help you write this test.
    • You are not allowed to send email, post to forums or communicate the details of your test using any other social media. If you are caught you will be asked to leave the lab and you will receive a mark of 0.
  • You may ask your lab instructor for help if you are having trouble with the hardware or Arduino IDE.
  • You may not ask your lab instructor for help with your sketch code.
  • You have the full 1h50m of your scheduled lab to complete the test. The upload window will stay open an extra 5 minutes, but no longer.

Problem Description

Use the Arduino and Danger Shield provided by your lab instructor to create a mystery box. The Arduino you received already has a sample mystery box sketch loaded onto it that responds to the code 5-1-8. You should watch the video below and read the instructions to understand how it should work.

Your solution should consist of a set up sketch, and a mystery box sketch

  • The set up sketch will store a 3 number code on the Arduino's EEPROM in any memory locations you wish.
    • The numbers may be hard coded into the sketch.
    • You may wish to retrieve and print the numbers to the serial monitor to confirm that they were stored correctly.
  • The mystery box sketch should work like this:
    • Start from one of these files:
      • Lab Test Template (click to download)
      • Beginner Lab Test Template (username and password required)
    • Start Mode: When the mystery box is reset it goes into Start Mode. Start mode ends if a touch is sensed on the Cap Sense pad. The two LEDs on pins 5 and 6 will indicate that the pad was touched. The mystery box will then go into Code Entry Mode.

    • Code Entry Mode: Once the mystery box is in Code Entry Mode, the user can use the sliders and the buttons below them to enter a three digit code. 
      • Pressing the button below a slider will show a number between 0 and 9 on the seven segment display corresponding to the slider's position.
      • When a button is released, a variable will remember a guess based on the last number shown while the button was being held.
      • When no buttons are being held, the seven segment display will show nothing.
      • If the three guesses match the three digits in the code that your set up sketch put on EEPROM, then the mystery box will go into Success Mode.

    • Success Mode: When the correct code has been entered and recognized in code entry mode, success mode will reveal the mystery box's secret.
      • Use a for loop to play back a sequence of letters from an array to the seven segment LED display. The letters should spell out your name. There should also be a blank character at the beginning and end your name.
        • Use this seven segment alphabet to design the letters (courtesy of Japanese wikipedia):
        • You may use the alternate characters in the bottom row if you wish.
      • When your name has been shown, reset the guess and switch back to Start Mode.

Submission

  • Write and upload at least two complete sketches to Schoology.
  • If you wish, you may save copies of your work at crucial stages on the marking scheme below. 
    • These copies may be submitted with your final solution to Schoology.
    • If something is missing from your final solution, the other copies will be examined for part marks. 
    • If you submit more than one copy of a sketch, please indicate which part of the marking scheme each copy corresponds to. A complete solution can be marked as final. For example, you might submit these files:
      • set_up_final.ino
      • mystery_box_4.ino (everything up to point 4)
      • mystery_box_7.ino (sketch showing that you know how to display your name)
      • mystery_box_final.ino (incomplete effort to solve mystery box problem)

Marking Scheme

  1. set up sketch 
    1. Stores three numbers in EEPROM (3 marks)
  2. mystery box sketch
    1. Has your name and student number at the top (1 mark)
    2. Does nothing until the Cap Sense pad is touched (LDR) (2 marks)
    3. Has a function that can display and clear the seven segment LED (1 mark)
    4. Turns on the LEDs on pins 5 and 6 (1 mark)
    5. Shows a number from 0 to 9 corresponding to at least one slider's position when the button below the slider is held (4 marks)
    6. Stores a number from 0 to 9 corresponding to at least one slider's position when the button below the slider is released (2 marks)
    7. Shows and stores numbers for all three sliders and buttons (2 marks)
    8. Displays your name from an array correctly (5 marks)
    9. Only displays your name after the full correct code has been entered. If you cannot display your name, a beep on the buzzer will do (3 marks)
    10. The mystery box changes modes as described above. ie: it does nothing until a touch is detected, then turns on the LEDs, then allows code entry, then if the correct code is entered it shows your name once and re-enters start mode. (1 mark)

Total Marks: 25
